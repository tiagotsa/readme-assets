# 🗃️ Central de Assets e Templates (README)

Este repositório serve como uma central de armazenamento de **imagens, ícones, GIFs, mockups e templates** utilizados para compor a documentação (`README.md`) de todos os meus projetos no GitHub.

## 💡 Por que este repositório existe?

A ideia principal é centralizar recursos visuais para:
- Manter o histórico de commits e o tamanho dos repositórios de código (aplicações) mais limpos e leves.
- Reutilizar facilmente as mesmas imagens em diferentes serviços (ex: frontend e backend do mesmo projeto).
- Padronizar a apresentação visual do meu portfólio.

## 📂 Organização das Pastas

Os arquivos estão organizados em pastas, separadas por projeto:

- `📁 /dmsystem` - Assets, telas e diagramas do projeto DM System.
- `📁 /iptv-premium` - Imagens de demonstração da plataforma de streaming IPTV.
*(Adicione ou atualize os nomes das pastas conforme for criando novos projetos)*

## 🔗 Como usar os links em outros projetos

Para referenciar as imagens daqui nos READMEs de outros repositórios:

1. Navegue até a imagem desejada dentro deste repositório.
2. Clique na imagem e depois clique em **"Raw"** (ou clique com o botão direito e escolha "Copiar endereço da imagem").
3. Certifique-se de que o link começa com `https://raw.githubusercontent.com/tiagotsa/readme-assets/...`
4. Use o link copiado no seu projeto de destino:

```html
<!-- Exemplo de uso em HTML para centralizar e controlar tamanho -->
<p align="center">
  <img src="LINK_COPIADO_AQUI" width="800px" alt="Descrição da imagem">
</p>
